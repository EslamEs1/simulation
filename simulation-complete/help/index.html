Getting Started

Welcome to the Pre-Authorization Management System! This guide will help you get started.

System Overview

The system allows healthcare providers to submit pre-authorization requests for medical tests and procedures. Reviewers can then evaluate and approve or reject these requests based on clinical criteria.

Key Features

  • Submit pre-authorization requests through a 7-step wizard
  • Track request status in real-time
  • Review and make decisions on requests
  • Manage tests and users (Admin only)
  • Generate reports and analytics

User Guides

How to Submit a Request

  1. Click "Create New Request" from your dashboard
  2. Select the medical test from the list
  3. Enter patient information
  4. Add clinical symptoms
  5. Enter ICD-10 diagnostic codes
  6. Provide clinical justification
  7. Upload supporting documents
  8. Review and submit

How to Review a Request

  1. Access the Review Queue from your dashboard
  2. Select a request to review
  3. Review all sections carefully
  4. Make a decision (Approve, Reject, Request More Info, or Transfer)
  5. Provide comments if needed
  6. Submit your decision

Frequently Asked Questions

How long does it take to get a decision?

Most requests are reviewed within 24-48 hours. Urgent requests are prioritized and may be reviewed sooner.

Can I edit a request after submission?

Once submitted, requests cannot be edited. However, if a reviewer requests more information, you can resubmit with updates.

What documents are required?

Required documents vary by test type. The system will indicate if documents are mandatory during the submission process.
